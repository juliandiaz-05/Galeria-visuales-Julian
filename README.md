# Galería de Personajes de Dragon Ball

## Descripción
Este proyecto es una **galería visual de 20 personajes de Dragon Ball**, creada como práctica de desarrollo web.  
Permite ver imágenes, nombres y una breve descripción de cada personaje, organizados en un **diseño de tarjetas en 5 columnas** con estilos modernos y colores personalizados.

---

## Tecnologías utilizadas
- HTML5
- CSS3
- Git & GitHub para control de versiones
